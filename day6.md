# Day 6 - File Comparison & Text Processing

cmp file1 file2        # Compare files
diff file1 file2       # Show differences
diff3 file1 file2 file3 # Compare 3 files
wc file.txt            # Count lines/words/chars
od file.txt            # File in octal format
locate filename        # Find file
tac file.txt           # Reverse file content
tee file.txt           # Output + save
sed 's/old/new/' file.txt # Replace text
more file.txt          # View file page by page
