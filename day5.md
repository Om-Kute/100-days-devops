# Day 5 - File, System Info & Text Processing Commands

## Commands

touch file.txt
# Create a new empty file

echo "Hello Linux"
# Print text or write into file

lsusb
# Show connected USB devices

lscpu
# Show CPU information

head file.txt
# Display first lines of file

tail file.txt
# Display last lines of file

sort file.txt
# Sort file content

sort -r file.txt
# Sort file content in reverse order

uniq file.txt
# Remove duplicate lines
