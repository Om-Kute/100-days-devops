Day 23 – OSI Model, TCP/IP Model & TCP vs UDP
Objective

Understand how data travels across a network and learn the difference between the OSI Model, TCP/IP Model, TCP, and UDP.

1. OSI Model (7 Layers)

The OSI (Open Systems Interconnection) model is a reference model developed by ISO to explain network communication in seven layers.

Layer	Name	Purpose
7	Application	Provides network services to user applications
6	Presentation	Data formatting, encryption, compression
5	Session	Manages sessions between applications
4	Transport	End-to-end communication, reliability, flow control
3	Network	Logical addressing and routing
2	Data Link	Node-to-node delivery and error detection
1	Physical	Transmission of bits through cables or wireless media
