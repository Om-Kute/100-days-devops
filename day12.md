# Day 12 - System Information & Checksum Commands

cksum file.txt
# Show checksum and byte count

sum file.txt
# Display checksum

md5sum file.txt
# Generate MD5 hash

showkey -v
# Show keyboard scan codes

stty -a
# Terminal settings

uptime
# System uptime and load

df -h
# Disk space usage

arch
# System architecture

hwclock -r
# Read hardware clock

free -h
# Memory usage
