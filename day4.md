# Day 4 - Calendar, Date & System Identity

1.cal 2026        # Show full year calendar
2.cal -3          # Show 3-month calendar
3.date            # Current date/time
4.date -s "2026-04-29 10:00:00"   # Set date/time
5.whoami          # Current user
6.who             # Logged in users
7.hostname        # System name
8.hostname -s     # Short hostname
9.tty             # Current terminal
